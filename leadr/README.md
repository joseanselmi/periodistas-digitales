# Leadr

Plataforma de clases online para alumnos de Periodistas Digitales.

## Que es

Leadr es donde los alumnos acceden a clases, prompts, automatizaciones y bonus.
El admin crea contenido con IA (Claude + ElevenLabs) y lo publica directamente.

## Planes

- **Basic — Periodista Digital**: acceso a contenido base
- **Pro — Periodista Digital Pro**: acceso completo a todo el contenido

## Archivos en esta carpeta

- `plan-lanzamiento.md` — plan completo de 10 dias para ir a produccion con tests por fase
- `prototipo.html` — prototipo clickeable completo (ver en ../prototipo.html)

## Stack

- Next.js + Supabase + Vercel
- Claude API para generacion de slides
- ElevenLabs para audio (modo v2)
- YouTube para videos embebidos
